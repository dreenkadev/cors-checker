# Cors Checker

CORS misconfiguration detector.

Pendeteksi miskonfigurasi CORS.


## Description

Tests web applications for CORS misconfigurations including wildcard origins and reflected origins.

## Deskripsi

Menguji aplikasi web untuk miskonfigurasi CORS termasuk wildcard origin dan origin tercermin.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Check: python cors_check.py https://target.com
- Demo: python cors_check.py --demo

## Penggunaan

- Periksa: python cors_check.py https://target.com
- Demo: python cors_check.py --demo
